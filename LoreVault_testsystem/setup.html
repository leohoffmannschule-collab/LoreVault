Chrome-Sicherheitseinschränkung

Einen kurzen Schritt
vor dem Lesen

Chrome blockiert den Zugriff auf lokale Dateien. Ein kleiner lokaler Server (1 Befehl) löst das dauerhaft.

Warum funktioniert es nicht direkt?

Wenn du index.html direkt im Browser öffnest (über file://), blockiert Chrome aus Sicherheitsgründen den Zugriff auf andere lokale Dateien – also auch auf deine PDFs im books/ Ordner.

Die Lösung: Ein minimaler lokaler Webserver, der die Dateien über http:// ausliefert. Python ist auf fast jedem Computer vorinstalliert und dafür perfekt geeignet.

Ordnerstruktur

So sollte dein Ordner aussehen:

📁 lorevault/
  ├── index.html
  ├── add-book.html
  ├── book-detail.html
  ├── reader.html
  ├── setup.html← diese Seite
  ├── start-server.bat← Windows
  ├── start-server.sh← Mac/Linux
  ├── 📁 css/
  ├── 📁 js/
  └── 📁 books/
      ├── meinbuch.pdf← deine PDFs hierher
      └── nocheins.pdf
Server starten
1
Python installiert?

Falls nicht: python.org/downloads – beim Installieren „Add Python to PATH" aktivieren.

2
start-server.bat doppelklicken

Die Datei liegt im LoreVault-Ordner. Doppelklick startet Server und öffnet den Browser automatisch.

3
Oder: manuell im Terminal

Öffne cmd/PowerShell im LoreVault-Ordner (Shift+Rechtsklick → Terminal öffnen):

python -m http.server 8000 kopieren
1
start-server.sh ausführbar machen (einmalig)
chmod +x start-server.sh kopieren
2
Script starten

Terminal im LoreVault-Ordner öffnen (cd ~/pfad/zu/lorevault):

./start-server.sh kopieren
3
Oder direkt:
python3 -m http.server 8000 kopieren
1
Python 3 installiert?
sudo apt install python3 kopieren
2
Script starten:
bash start-server.sh kopieren

Oder direkt:

python3 -m http.server 8000 kopieren

Terminal/Kommandozeile im LoreVault-Ordner öffnen und einen dieser Befehle eingeben:

Py

LoreVault öffnen (localhost:8000) ← Erst nach dem Serverstart klicken
Tipp: Lesezeichen setzen

Setze http://localhost:8000 als Lesezeichen in Chrome. In Zukunft reicht es, den Server zu starten und dann das Lesezeichen zu öffnen – du musst nie wieder manuell navigieren.
